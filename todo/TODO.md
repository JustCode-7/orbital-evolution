4. Capacitor in der finalen Version nutzen für NativeApps
  - Installation: npm install @capacitor/core @capacitor/cli
  - Initialisierung: npx cap init
  - Plattform hinzufügen: npx cap add android oder npx cap add ios
  - synchronisieren: Nach jedem Build deiner Angular-App (npm run build) führst du npx cap copy aus, um die Web-Assets in die nativen Ordner zu schieben.
